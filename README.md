# AI测试助手

运用AI技能提效测试的个人网站

## 简介

这是一个关于AI在软件测试领域应用的个人网站，提供在线测试工具和博客文章。

## 功能特性

- 📝 智能测试用例生成器
- 🧠 需求转XMind思维导图
- 🔤 正则表达式生成器
- 🗄️ SQL查询生成器
- 🌐 API文档生成器
- 📧 测试报告模板生成

## 技术栈

- HTML5
- CSS3 (响应式设计)
- JavaScript (原生)
- 无需后端，纯静态网站

## 部署

网站已配置为可部署到 GitHub Pages：

1. 将代码推送到 GitHub 仓库
2. 进入仓库设置
3. 启用 GitHub Pages
4. 选择 main 分支作为源

## 在线访问

部署后可通过 `https://liuliu3198.github.io/ai-test-helper` 访问
