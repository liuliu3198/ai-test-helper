# AI测试助手 - 完整使用指南

## 🚀 快速开始

### 方式一：使用本地生成器（推荐先用这个测试）

1. 打开网站（http://localhost:8080）
2. 点击任意工具卡片
3. 点击"本地生成"按钮
4. 即可获得智能生成的测试内容

**无需任何配置，所有功能立即可用！**

### 方式二：启动后端服务（需要API Key）

#### 步骤1：获取免费API Key

1. 访问 [硅基流动](https://siliconflow.cn) 注册账号
2. 进入控制台 → API密钥 → 创建新密钥
3. 复制API Key

#### 步骤2：配置后端

```bash
# 1. 进入后端目录
cd ai-test-helper/backend

# 2. 安装依赖
npm install

# 3. 配置API Key
# 编辑 .env 文件，填入你的API Key
SILICON_API_KEY=你的API密钥

# 4. 启动后端服务
npm start
```

#### 步骤3：配置前端

1. 打开网站 http://localhost:8080
2. 点击任意工具卡片
3. 点击"⚙️ API设置"按钮
4. 在"后端服务地址"填入：`http://localhost:3001`
5. 点击"保存配置"
6. 点击"测试后端"确认连接成功

#### 步骤4：使用AI生成

1. 输入需求内容
2. 点击"生成内容"按钮
3. AI将自动生成专业的测试内容

## 📱 功能说明

| 功能 | 说明 |
|------|------|
| 测试用例生成 | 输入需求，生成完整测试用例表格 |
| 需求转XMind | 将需求整理为思维导图结构 |
| 正则表达式生成 | 描述需求，自动生成正则表达式 |
| SQL查询生成 | 自然语言描述转SQL语句 |
| API文档生成 | 接口描述转完整API文档 |
| 测试报告模板 | 测试情况转专业测试报告 |
| 接口测试 | 在线发送HTTP请求测试API |
| 测试数据生成 | 生成姓名、手机号、邮箱等 |
| JSON格式化 | 格式化、校验、压缩JSON |
| 代码转换 | JSON转Java/TypeScript/Go/SQL |
| 编码转换 | URL/Base64/MD5转换 |
| 时间戳转换 | 时间戳与日期互转 |

## 🔧 故障排除

### 本地生成器不工作
- 检查浏览器控制台是否有错误
- 刷新页面重试

### 后端服务连接失败
- 确认后端已启动（npm start）
- 确认端口3001未被占用
- 检查API Key是否正确配置

### AI生成失败
- 检查API Key是否有效
- 检查网络连接
- 查看后端控制台错误日志
